# 🧹 Limpeza e Tratamento de Dados – Churn

Este projeto apresenta um exercício de limpeza e tratamento de dados utilizando Python, aplicado a uma base de dados relacionada a churn de clientes.

O objetivo é preparar os dados para análise, realizando etapas fundamentais de data cleaning, como identificação de inconsistências, tratamento de valores ausentes e padronização das informações.

## 🎯 Objetivo do Exercício

Identificar problemas na base de dados

Tratar valores nulos ou inconsistentes

Padronizar formatos de dados

Preparar o dataset para análise ou modelagem

Essas etapas são essenciais no fluxo de trabalho de Ciência de Dados.

## 📊 Etapas Realizadas

Durante o notebook são realizadas atividades como:

Carregamento do dataset

Inspeção inicial da base de dados

Identificação de valores nulos

Tratamento de dados inconsistentes

Conversão de tipos de dados

Limpeza e preparação do dataset

## Importância 

Essa etapa é essencial para projetos de análise de dados e machine learning.
